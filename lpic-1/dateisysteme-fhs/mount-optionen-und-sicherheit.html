Mount‑Optionen & Sicherheit

Lernziele

Cheatsheet

# Häufige Optionen
nodev,noexec,nosuid,ro,noatime,uid=,gid=,umask=
# Beispiele (fstab)
/dev/sdb1 /srv ext4 defaults,nodev,noexec,nosuid 0 2
UUID=...   /media/usb vfat rw,uid=1000,gid=1000,umask=027 0 0

Theorie

  • noexec: Verhindert das Ausführen von Binärdateien auf dem Mount. Skripte können weiterhin via Interpreter laufen.
  • nosuid: Ignoriert setuid/setgid‑Bits → reduziert Privilege‑Escalation‑Risiken.
  • nodev: Unterbindet das Anlegen/Verwenden von Device‑Files im Mount.
  • noatime/nodiratime/relatime: Steuern Zugriffszeitstempel; relatime ist Standard (Performance vs. Compliance).
  • ro/rw: Read‑only/Read‑write Mount; ro sinnvoll für Backups/forensische Zwecke.
  • vfat/ntfs: Keine Unix‑Rechte → uid=, gid=, umask= bestimmen effektive Rechte.
  • hidepid (procfs): hidepid=2 verbirgt fremde Prozesse; gid=proc für Monitoring‑Gruppe.
  • systemd‑Integration: x-systemd.automount, nofail, x-systemd.idle-timeout verbessern Boot‑Robustheit und On‑Demand‑Mounts.

Praxis

# Temporär mit Sicherheitsschaltern mounten
sudo mount -o nodev,nosuid,noexec /dev/sdb1 /srv

# fstab: USB‑Stick (vfat) mit festen Besitzern
UUID=XXXX  /media/usb  vfat  rw,uid=1000,gid=1000,umask=027,noexec,nodev,nosuid  0  0

# proc absichern
proc  /proc  proc  defaults,hidepid=2,gid=proc  0  0

# Netzwerkdateisystem robust einhängen
server:/share  /mnt/share  nfs  rw,_netdev,nofail,x-systemd.automount,x-systemd.idle-timeout=60s  0  0

# atime reduzieren
/dev/nvme0n1p3  /var  ext4  rw,relatime  0  2

Best Practices

  • Weiche Pfade vermeiden: Für fstab nach Möglichkeit UUID= oder LABEL= nutzen.
  • Least Privilege: Für Wechseldatenträger stets nodev,nosuid,noexec setzen.
  • Boot‑Stabilität: Externe/NFS mit nofail, _netdev und ggf. Automount versehen.
  • Auditing: Kritische Mounts dokumentieren und regelmäßig mit findmnt/mount prüfen.

Erweitertes Cheatsheet

# Aktive Mount‑Optionen anzeigen
findmnt -no TARGET,OPTIONS /srv

# Bind‑Mount mit noexec
mount --bind /opt/app/bin /sandbox/bin
mount -o remount,bind,noexec /sandbox/bin

# tmpfs abgesichert
tmpfs /run/tmpfs tmpfs rw,nosuid,nodev,mode=0755,size=256M 0 0

Übungen

  1. Hänge ein Verzeichnis als ro,nodev,nosuid,noexec ein und verifiziere das Verhalten.
  2. Erzeuge einen sicheren vfat-Mount für einen USB‑Stick mit umask/uid/gid.
  3. Setze hidepid=2 für /proc und teste mit einem Nicht‑Privilegierten Benutzer.
