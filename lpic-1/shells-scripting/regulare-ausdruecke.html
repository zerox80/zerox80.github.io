Syntax-Überblick

^foo$          # exakte Zeile
[0-9]{3,}      # mind. 3 Ziffern
colou?r       # optionales u
([A-Za-z_]+)  # Gruppe 1 (Wort)
\1            # Backreference auf Gruppe 1
\s+           # Whitespace (sed/grep -P nötig)

Viele Tools unterscheiden zwischen Basic und Extended Syntax. Nutze grep -E bzw. sed -E für +, ?, | ohne Backslashes.

Beispiele

# E-Mail grob erkennen
grep -E "^[^@]+@[^@]+\.[^@]+$" users.txt

# Zeilen mit doppeltem Wort (Backref)
grep -E "\b([A-Za-z]+)\s+\1\b" text.txt

# sed: Ersetzen mit Gruppenreferenz
sed -E 's/(server_name)\s+.*/\1 example.org/' nginx.conf

Übungen

  1. Extrahiere alle IPv4-Adressen aus einer Datei.
  2. Finde doppelte Wörter in einem Text (z.B. „the the“).
  3. Setze in einer Config den Port-Wert auf 8080 nur, wenn eine server-Zeile folgt.
