Telemetry minimieren PowerShell Admin

Setzt Telemetrie-Werte auf Minimum, deaktiviert ausgewählte Tasks/Dienste. Empfohlen: zuerst Backup/Wiederherstellungspunkt.

Download .ps1 Download .cmd Revert .ps1 Revert .cmd 
.\assets\scripts\telemetry_minimize.cmd
powershell -ExecutionPolicy Bypass -File .\assets\scripts\telemetry_minimize.ps1
powershell -ExecutionPolicy Bypass -File .\assets\scripts\telemetry_minimize.ps1 -DisableWER
.\assets\scripts\telemetry_revert.cmd
powershell -ExecutionPolicy Bypass -File .\assets\scripts\telemetry_revert.ps1
Get-FileHash .\assets\scripts\telemetry_minimize.ps1 -Algorithm SHA256

Sichere Defaults REG

Importiert empfehlenswerte Registry-Defaults (Telemetrie minimiert, Empfehlungen/Ads aus, Bing/Cortana aus).

Download .reg Revert .reg

reg import .\assets\scripts\revert_defaults.reg
Get-FileHash .\assets\scripts\privacy_defaults.reg -Algorithm SHA256

Optionale Bloatware entfernen PowerShell Admin

Listet vorab alle Kandidaten (Vorschau). Mit -Apply wirklich entfernen. Vorsichtig einsetzen.

Download .ps1 Download .cmd 
.\assets\scripts\remove_bloat.cmd
powershell -ExecutionPolicy Bypass -File .\assets\scripts\remove_bloat.ps1
powershell -ExecutionPolicy Bypass -File .\assets\scripts\remove_bloat.ps1 -List
powershell -ExecutionPolicy Bypass -File .\assets\scripts\remove_bloat.ps1 -Apply -CurrentUserOnly
powershell -ExecutionPolicy Bypass -File .\assets\scripts\remove_bloat.ps1 -Apply
powershell -ExecutionPolicy Bypass -File .\assets\scripts\remove_bloat.ps1 -Apply -ProvisionedOnly
Get-FileHash .\assets\scripts\remove_bloat.ps1 -Algorithm SHA256
